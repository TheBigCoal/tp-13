# tp-13
Implementación de Base de Datos y Aplicación de Consola
Este trabajo práctico consiste en implementar la base de datos del proyecto ABP en Clever Cloud y desarrollar una aplicación de consola en Python que permita realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre al menos dos tablas del modelo.

El objetivo del proyecto es demostrar dominio en:

Modelado de base de datos

Conexión y gestión de una BD MySQL

Programación básica en Python con acceso a datos

Implementación de un menú interactivo por consola
